1. What is HTML?

HTML is the HyperText Markup Language used to describe web pages.

2. How many versions of HTML do you know?

HTML (1991): The first version of HTML, proposed by Tim Berners-Lee.

HTML 2.0 (1995): The first standard version of HTML, introducing features like forms and tables.

HTML 3.2 (1997): Added support for features like tables, applets, text flow around images, and internationalization.

XHTML 1.0 (2000): An XML-based version of HTML 4.01, designed to be more strict and compatible with XML.

HTML5 (2014): The latest major version of HTML, introducing new features like native support for video and audio, canvas for drawing, and improved support for web applications.

3. What is the general structure of a typical web page?

DOCTYPE declaration, HTML tags, head and body sections, and various content elements like headers, paragraphs, links, and images.

4. Could you describe the constituting parts of the Web page? Open your favorite website in your browser and try to find these parts in it.

Header - the header typically contains branding elements such as the website logo, navigation menu, search bar, and sometimes contact information or social media links.

Nav/Navigation - this part consists of menus or links that help users navigate through the website. It may be part of the header or appear separately.

Content - the main content area of the webpage where text, images, videos, and other multimedia elements are displayed. This section can vary significantly depending on the type of website.

Sidebar - sidebars often accompany the main content area and contain supplementary information, links to related content, advertisements, or additional navigation elements.

Footer - the footer typically appears at the bottom of the webpage and contains copyright information, legal disclaimers, links to important pages (such as privacy policy or terms of service), and additional nav links.

5. What are HTML elements? What is the difference between opening and closing tags?

An opening tag is the first part of an HTML element and is enclosed in angle brackets < >. It contains the name of the element.

A closing tag is the second part of an HTML element and is also enclosed in angle brackets < >. It contains the name of the element preceded by a forward slash /.

An HTML Document is mainly divided into two parts: HEAD: This contains the information about the HTML document. For Example, the Title of the page, version of HTML, Meta Data, etc. BODY: This contains everything you want to display on the Web Page.
